## 冲突背景 / Conflict Background

2026年2月28日，美国和以色列对伊朗发动联合空袭（代号Epic Fury行动），摧毁军事设施和核设施，导致伊朗最高领袖哈梅内伊死亡。伊朗以导弹和无人机袭击美军基地、以色列及海湾国家进行报复。

伊斯兰革命卫队（IRGC）随即宣布封锁霍尔木兹海峡。截至3月中旬，伊朗已对至少21艘商船发动袭击。全球约20%的石油贸易通过该海峡运输。布伦特原油价格飙升突破每桶126美元。

## 最新发展 / Latest Developments (截至3月26日)

### 停火谈判
- 美国通过巴基斯坦向伊朗递交15点和平方案，包括：核计划放弃、制裁解除、重开海峡
- 伊朗3月25日正式拒绝美方方案，提出5点反提案：
  1. 停止一切"侵略和暗杀"行为
  2. 建立具体机制确保战争不再重演
  3. 战争赔偿和补偿
  4. 所有战线全面停火（包括抵抗组织）
  5. 伊朗对霍尔木兹海峡行使主权
- 伊朗外长阿拉格齐称"没有与美国的对话"，拒绝谈判
- 特朗普威胁若海峡未开放将摧毁伊朗电厂，但随后推迟了最后期限

### 军事态势
- 以色列宣布击杀IRGC海军司令坦格西里（负责海峡封锁行动）
- 美国向中东增派2,500名海军陆战队员和1,000名第82空降师伞兵
- 3月19日起美军开始以低空战机和直升机巡逻海峡，清除伊朗无人机
- 伊朗国防委员会警告：任何对伊朗海岸的攻击将触发海湾水域布雷
- 伊朗继续向以色列、巴林、科威特、卡塔尔、沙特、阿联酋发射导弹

### 航运现状
- 海峡基本处于"物理畅通但功能封锁"状态
- 伊朗选择性允许"非交战国"船只通过：土耳其、印度、中国船只已通行
- 超过1,000艘船（多为油轮）滞留海峡外，约20,000名海员受困
- 战前每日通行量约100+艘，目前远低于10艘

## 关键利益方分析 / Key Stakeholder Analysis

### 美国 (Trump Administration)
- 目标：迫使伊朗投降，重开海峡，消除核能力
- 杠杆：压倒性军事力量，但国内舆论分歧（60%认为缺乏明确计划）
- 压力：油价暴涨影响中期选举前景，化肥短缺威胁农业州共和党选情
- 立场：愿意谈判但坚持最大化条件

### 伊朗
- 策略：横向升级（扩大冲突面积），利用海峡封锁制造经济压力迫使对手退让
- 能力：导弹和发射器储备正在消耗，战争第一周后发射速率已下降
- 意愿：拒绝任何被视为投降的协议，认为"仅需撑过轰炸就是胜利"
- 内部：最高领袖已死，权力结构不确定，可能影响决策

### 以色列 (Netanyahu)
- 立场：称伊朗已被"摧毁"，但认为政权更迭需要"地面部分"
- 同时扩大对黎巴嫩真主党的军事行动
- 战略目标与美国存在差异

### 海湾国家 (GCC)
- 沙特、阿联酋、卡塔尔等均遭受伊朗攻击
- GCC秘书长称伊朗"越过了所有红线"
- 但不愿直接卷入冲突

### 中国与印度
- 中国作为伊朗战略伙伴，船只已获准通行
- 印度与伊朗协商更多船只通行
- 两国立场可能影响海峡部分开放的可能性

## 历史基准率 / Historical Base Rates

- 1980年代两伊战争"油轮战争"：海峡从未完全关闭，但持续数年
- 1987-88年美伊护航战：数月内逐步恢复
- 2019年海峡紧张局势：未实质关闭，数周内缓解
- 当前局势远超以往任何先例，被称为"自1970年代以来最大能源供应中断"
